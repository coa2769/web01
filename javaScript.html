Javascript

[기본적인 내용]
사용자와 상호 작용하기 위해 만들어진 언어. javascript는 동적이다.(html은 정적이다.) ex) html에서 1+1을 쓰면 그래도 출력되지만 javascript에서는 1+1이 계산되어 2로 출력된다.
웹브라우저 >> 마우스 오른쪽 버튼 >> 검사 로 개발자 도구를 켤 수 있고 ESC를 누르면 나오는 console html, javascript등을 지금 페이지에 바로 적용 가능하다.
HTML의 주석문은 < !-- 내용 -- >이고 JS는 //이다.

1) html문서에서 javascript사용하기

CSS와 마찬가지로 어디서부터 javascript언어인지 웹브라이저에게 알려줘야한다. JS를 이용하여 제어하고자 하는 tag를 selector(선택자, CSS참고)를 이용하여 선택하고 디자인에 관련한 속성은 CSS를 통하여 작용할 때 마다 CSS코드를 JS가 삽입해준다.
- script, /script : 이 태그 안에 작성된 코드들은 javascript로 분석해 달라고 요천하는 태그이다. body태그 안에 들어간다.
- 태그의 Event atrribute(이벤트 속성)에서 ""안에 javascript내용을 넣는다.
- 검색어 : Javascript select tag by CSS selector (JS를 이용하여 제어하고자 하는 tag 찾을 때)
- 검색어 : Javascript element style (CSS코드를 어떻게 JS로 넣는가?)

2) 기본 문법

(1) Event attribute(이벤트 속성)

사용자가 Web Page와 자신가 맞는 작용을 하면 JS코드를 기억하고 있다가 실행해 준다.
- 여기에는 onclic, onchage, onkeydown 등이 있다.
- 검색어 : Javascript (작용 내용) event attribute
- input : 이 tag를 사용하면 버튼(button), 입력창(text)을 등을 만들 수 있다. input의 type속성에 생성하고 싶은 객체를 넣으면 된다.
ex)
< input type = "button" value = "hi" onclick = "aler('hi')" >
//value속성에 값을 넣으면 버튼의 이름을 넣을 수 있다.
< input type = "text" onchange = "alert('changed')" >
//onchage : 문자 입력 후 엔터 작용, aler() : 브라우저의 경고창을 띄워준다. 괄호 안에 경고 내용을 넣을 수 있다.
< input type = "text" onkeydown = "alert('key down!')" >
//onkeydown : 키 입력 작용

(2) Data type

javascript는 변수를 사용할 수 있고 입력하는 값들에 데이터 타입이있다. 또한 여러 연산자들을 사용할 수 있다.
단,변수에는 Data type이 없어서 아무 데이터나 입력해도된다.
- 검색어 : Javascript Data type
- 상수 -> 산술연산이 가능하다. ex) 10
- 문자열 -> 문자 데이터이다. Javascript string이라고 검색해보아라 여러 처리를 도와주는 함수 와 Properties를 찾아볼 수 있다. 문자열에서 +연산자는 두 문자열을 결합해준다.
ex) "문자 데이터"
'문자 데이터'
"문자 데이터".length //문자 개수를 알려주는 property
"1" + "1" -> "11"
** document.write() : web page에 글씨를 출력할 때 쓰인다.
ex) document.write("문자" + '< br >'); //이렇게 태그를 적용할 수도 잇다. document.write('< div style="color:red" >'+ "문자열" + '< /div >'); //CSS코드도 동일

(4) 반복문

C/C++코드 처럼 사용 가능하며 while문과 for문이 있다.
//=============================================================================

< while문 >

//document.querySelector('a')는 첫번째 태그만 가져온다. (그런 특성을 가지고 있음)
//여러 태그를 선택하려고 할때 => 검색어 : javascript get element by css selector multiple
//document.querySelectorALL('a') 모든 a태그를 배열처럼 가져온다.(하지만 실제 배열은 아니다.)
var alist = document.querySelectorAll('a');

var i = 0;
//반복문의 시작, 괄호안에 조건문을 넣어준다.
while(i < alist.length)
{
//개발자 도구의 console에서 실행창에 값이 출력 된다.(web page에 보이지 않음)
console.log(alist[i]);
//배열처럼 사용 가능
alist[i].style.color = 'powderblue';
//count에 쓰이는 변수의 값을 증가 시킨다.
i=i+1;
}
//=============================================================================

//반복해서 실행할 코드

(6) 객체

순서 없이 데이터를 저장해준다. 키 값으로 데이터를 꺼내올 수 있다.(이름으로 정리정돈된 상자)
객체에는 문자열, 숫자, 배열 등의 테이터와 함수(Method)를 담을 수 있다.

- 객체 생성

//coworkers라는 객체에 이름을 붙여서 데이터 저장
//배열같이 정보를 넣을 때는 대괄호, 객체같이 정보를 넣을 때는 중괄호
var coworkers = {
//키값/변수 : 데이터 또는 property : 데이터
"programmer" : "egoing",
"designer" : "leezche"
};

- 객체에 접근

// . : 객체에 접근하는 기호
//객체에서 데이터 가져오기
document.write("programmer : " + coworkers.programmer + "< br >");
document.write("designer : " + coworkers.designer + "< br >");

- 객체에 정보 추가

coworkers.bookeeper = "buru";
//띄어 쓰기가 키 값에 있을 때
coworkers["data scientist"] = "taeho";
document.write("bookeeper : " + coworkers["data scientist"] + "
");

- Iterate

객체 + 반복문에 관한 내용이다.
검색어 : javascript object iterate
//Key에는 키 값이 들어간다.
for(var key in coworkers)
{
document.write(key + ' : ' + coworkers[key] + '
');
}

- Property & Method

//C++의 클래스와 같이 이렇게 함수를 객체에 담을 수 있다.

//Method : 객체에 소속된 함수
//Property : 객체의 키 값들
coworkers.showAll = function(){
//단 showAll도 여기에 소속된 데이터이기 때문에 출력 된다.
for(var key in this)
{
document.write(key + ' : ' + this[key] + '< br >');
}
}
//객체의 함수 호출
coworkers.showAll();
