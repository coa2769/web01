CSS

[기본적인 내용]
HTML의 부족한 시각적 부분을 또 다른 언어인 CSS를 만들어 체웠다.(분업화) 중복되는 부분을 재거 할 수 있다 이런 중복 제거는 유지보수와 가독성을 높일 수 있다.

1) HTML에서 CSS언어 사용하기

Web browser는 기본적으로 코드들을 HTML코트라고 생각하고 번역한다.
그러므로 CSS코드를 쓸 때에는 web broser에게 이 코드가 CSS라는 것을 알려줘야한다.
; 은 한 효과에 대한 구분으로 반드시 Declaration뒤에 넣어주어야 한다.

- style, /style : 이 태그 안에 작성된 코드들은 CSS언어로 분석해 달라고 요청하는 태그이다.
head태그 안에 들어가서 본문의 전체 내용에 관해서 시각적 변경사항을 작성하는 부분이다.

2) 태그를 선택하는 선택자

- 태그{} : Selector(선택자). 이 페이지에 주고 싶은 효과를 누구에게 줄것인가를 선택해준다. {}안에는 효과에 대한 내용이며 이를 Declaration(선언)이라고한다.

ex)
//본문의 모든 a 태그에 대한 글들의 글자색을 빨간색으로 변경해달라는 코드
< style >
a{
color:red;
text-decoration: none;
}
< /style >

- HTML태그에 style이라는 속성을 사용한다.
이 속성은 반드시 값으로 CSS의 효과가 들어온다. style태그가 직접 들어가게 되면 선택자가 필요 없어진다.
ex) < a href="CSS_study.html" style="color:red;text-decoration: none;"> CSS < /a>

- HTML태그에 class라는 속성을 사용하여 그룹으로 CSS값 주기
head의 style태그안에 class선택자를 만들고 body에서 쓰이고 싶은 곳에 class속성을 넣어 사용한다. class는 다중 선택 가능하지만 보다 더 아래에 있는 class가 더 큰 영향력을 주게된다.
ex)
< head>
< style>
.saw {
color: gray;
}
.active {
color:red;
}
< /style>
< /head>
< body>
< a href="1.html" class="saw"> HTML < /a>
< a href="2.html" class="saw active"> HTML < /a>
< /body>

- HTML태그에 id라는 속성을 사용하여 그룹으로 CSS값 주기
head의 style태그안에 id선택자를 만들고 body에서 쓰이고 싶은 곳에 id속성을 넣어 사용한다. class선택자와 같이 사용하게 되면 id선택자가 최우선시 된다.(id선택자는 선언된 html에서 한번만 사용가능하다.)
ex)
< head >
< style >
#active {
color:red;
}
.saw {
color:gray;
}
< /style >
< /head >

< a href="2.html" class="saw" id="active" > CSS < /a >

** (우선순위)id선택자 > class선택자 > tag선택자
** 같은 선택자 끼리는 더 밑에 써져있는 것이 우선순위가 높다
** tag선택자로 전체적 디자인을 선택하고 예외적인 것을 id로 디자인 한다.
** 같은 내용의 속성을 태그에 적용하게 된다면 선택자의 태그를 여러개 쓸 수 있다.
** 같은 종류의 Property는 값을 연속으로 나열하면서 한번에 설정 가능하다.
** #(아이디 이름) [태그]를 선택자로 쓰면 저 아이디 밑에 있는 어떤 태그에 내용을 적용하라는 뜻이다.
ex)
h1, a{
border-width : 5px;
border-color : red;
border : 5px red; //위 내용을 한번에 나타내는 방법
}

2) 여러 CSS언어

- color:색깔
색깔의 설정해준다.
ex) color:black, color:red
- text-decoration:
텍스트의 꾸밈에 대해 설정해준다.
ex) text-decoration: none
- font-size:
글자의 크기를 설정할 수 있다.
ex) font-size: 50px;
- text-align: 글자의 정렬을 정해준다.
ex) text-align: center;
- border-width : 태그의 테두리의 넓이를 정해준다.
ex) border-width : 5px;
- border-color : 태그의 테두리의 색깔을 정해준다.
ex)border-color : red;
- border-style : 태그의 테두리의 디자인을 정해준다.
ex)border-style : solid;
- display : 화면 상에 보여주는 것들을 바꿀 수 있다.
ex) display : inline
block level element를 inline element로 바꿔준다.
display : none
태그의 영향을 받지 않은 글을 볼 수 있다.

3) 박스모델

html태그 하나하나를 박스로 생각하여 부피가 존재하도록 되어있다.

- padding :
이 설정은 content를 둘러싼 테두리안의 크기를 조정할 수 있다.
ex) padding:20px;
- margin :
두 태그의 테두리를 기준으로 한 여백이 없어진다.
ex)margin:20px;
- width :
content의 크기를 조정할 수 있다.
ex) width:100px;

4) 그리드

- span,/span : 이 태그는 div와 같은 기능을 한다. (단,이 태그는 inline element이다.)

5) 반응형디자인

화면의 크기에 따라서 웹페이지 각 요소들 반응해서 동작하는 것을 말한다. 이것을 CSS에서 구현하기위해 Media query를 사용한다.
- Media Query : 이것은 어떤 조건에 만족 했을 때만 CSS내용 동작하는 것을 기능을 제공한다.
//min-width : (픽셀값) => screen width > (픽셀값)
//max-width : (픽셀값) => screen width < (픽셀값)
@media(max-width:800px) {
div{
display:none;
}
}

6) CSS파일 재사용

같은 CSS내용을 다른 페이지에 똑같이 적용할 때 이것이 여러개 있다면 작업하는데 효율이 나쁘다. 그러므로 확장자가 .css인 파일에 공통으로 쓰이는 CSS에 대한 내용을 넣어서 각 html파일에 이것을 적용하는 방법으로 중복을 줄이려한다.
- link : 이 태그를 사용하여 이 파일이 어떤 CSS파일과 연결되어 있는지를 웹브라우저에게 알려줄 수 있다.
ex) < link rel="stylesheet" href="style.css">
