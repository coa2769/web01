Vue.js
========================================================================
vue.js devtools : chrome에서 vue개발에 필요 개발도구 
이 책에서는 패키지 관리를 npm대신 yarn으로 한다.
vue-cli : vue의 프로젝트를 생성과 기본 폴더 설정을 해주는 모듈.
```
npm install -g yarn @vue/cli
```

vue-cli으로 프로젝트 생성
```
//commainline으로 생성
vue create [프로젝트 폴더 명]

//GUI로 실행되어
vue ui

//GUI에서 실행 방법
작업목록 >> serve >> 실행 >> 앱열기
```
